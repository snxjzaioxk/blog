# c语言笔记

## 变量和常量

### 变量

可以改变的值

### 常量

1.即数字不会改变的值

2.用const来定义常量  也就是变量不变



## 运算符优先级

### 单目运算符

+单目不变 自右向左

举例 a*+b

-单目取负 自右向坐

举例a*-b

### 双目运算符

+-*/%

自左向右

### 赋值运算符

=

自右向左

### 增增减减运算符

a++ 是a加1以前的结果

++a是a加1以后的结果

## for循环 

for（初始动作；条件；每轮动作）{

}

for循环中每一个表达式都可以省略的

for（；条件；） == while（条件）

### 三种循环使用场景

如果有固定次数 用for

如果必须执行一次 用while

其他情况 用while
